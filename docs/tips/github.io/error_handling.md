# Error Handling for GitHub Pages

## Error

```bash
Error: The process '/opt/hostedtoolcache/Ruby/3.2.2/x64/bin/bundle' failed with exit code 16
```

### Solution
```bash
bundle lock --add-platform x86_64-linux
```
